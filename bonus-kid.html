Treat Your Children

Bonus Kid  is an app that lets parents give incentives to their kids. The parent can add tasks and rewards to each child they have. The child can send a response saying they have completed the task or buy a reward from the store the parent has made. This app uses React, MongoDB, Express, and mocha, chai, enzyme and, jest.

 

CHALLENGES

The biggest challenge was having a parent to child relationship in the backend:

  • We had to create a parent model and have it reference the child
  • We had to create a child model and have it reference the parent

After finishing the parent and child relationship the front end was making to many calls to the backend:

  • The client side had to create a for loop to call the rewards, children and tasks
 

SOLUTIONS

We started with a parent model, added the functionality it needed We changed the whole backend to just send authTokens. Each endpoint was updated to update the database and give back an updated authToken.

 

RESULTS

The end result was a working app that allowed a parent to have children. The parent is able to add tasks and rewards to specific children. The child can submit that a task and buy something from the shop if they have enough points. This app was made in a group, you can check them out here.
